# 鸣潮国际服抽卡分析工具

> :warning:注意：
>
> 1. 实验性产物，谨慎使用。理论上无风险，如产生意外概不负责。
> 2. 开发平台为Windows 11，其他平台是否适用以实际情况为准。
> 3. 注意是国际服的抽卡分析，国服未测试，是否适用以实际情况为准。

鉴于库街区、小黑盒等工具都不适用国际服，所以自己动手做了一个简陋的分析工具。目前该在本地运行正常，但没有经过其他平台测试，适配性有待验证。



## 一、环境需求 & 开发背景

- **运行**

  1. 操作系统：Windows 11 (64-bit)

  2. 拥有鸣潮（国际服）游戏本体，在**运行前打开游戏并查询一次历史记录**

     > 由于工作原理需求鸣潮的运行日志，所以显示的历史记录查询行为是在每次同步记录前必须的

- **二次开发**
  1. Go
  2. HTML

## 二、基本介绍

核心是通过抓取鸣潮运行产生的日志文件，找出其中获取抽卡历史记录的请求URL并解析。通过本地再请求获得原始数据并分析，生成HTML可视化分析报告。
